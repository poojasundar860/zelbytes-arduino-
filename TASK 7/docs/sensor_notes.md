# Sensor Notes

## Sensors Used

* DHT22 Temperature & Humidity Sensor
* Soil Moisture Sensor
* LDR (Light Sensor)

## Working

The Arduino reads sensor values at regular intervals and logs them in CSV format through the Serial Monitor. The data includes timestamp, temperature, humidity, soil moisture, and light intensity.

## CSV Format

timestamp,temp,humidity,soil,light

## Observation

Temperature and humidity remained relatively stable, while soil moisture and light values varied according to environmental conditions.
